# CPSC_481_Capstone

## Roles
### 1. Systems Architect (Offensive Security Simulation)

Responsible for designing the overall architecture of VectorForge, including modular system structure, data flow between components, and integration of the simulated environment with the agent. Defined how reconnaissance, state representation, and decision-making components interact as a cohesive system. Ensured clear separation of concerns and maintainability while supporting the implementation of an intelligent agent operating within a simulated network.

---

### 2. Agent Logic and Decision System Engineer (Mánu Uribe)

Responsible for implementing the core decision-making system of the penetration testing agent. Designed the state space representation of the environment, defined possible actions, and implemented A* heuristic search to evaluate and select optimal attack paths. Developed logic for prioritizing actions based on discovered information, estimated outcomes, and environmental constraints such as vulnerability severity and access requirements.

---

### 3. Evaluation and Analysis Engineer

Responsible for designing evaluation metrics and analyzing the performance of the agent across different simulated network scenarios. Measured efficiency based on factors such as time to discovery and vulnerability importance. Collected and interpreted results to assess how effectively the agent identifies high-value targets, and provided insights to refine the decision-making system.
